# Contributing

Thanks for contributing to Mucaro Scout.

## Local dev

1. `make up`
2. `make setup-index`
3. `make ingest-sample`

## Contribution rules

- Keep PRs small and focused.
- Add or update docs when behavior changes.
- Preserve canonical field model unless discussed in an issue.
- Run smoke checks before opening a PR.

## Suggested roadmap issues

- Additional parsers (Zeek, Suricata, Windows EVTX exports)
- Better batching and backpressure controls
- Saved searches and dashboard presets for class labs
- RBAC and class/team tenancy
