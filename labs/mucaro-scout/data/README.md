# Mucaro Scout data pack

Scenario: suspicious domain investigation around `sfrclak.com` (`142.11.206.73`).

Files are available in `suspicious-domain/`:

- `dns.jsonl` - 100 DNS events
- `proxy.jsonl` - 100 proxy events
- `firewall.jsonl` - 100 firewall events
- `auth.jsonl` - 100 authentication events
- `endpoint.jsonl` - 100 endpoint/process events
- `combined.jsonl` - 500 merged events sorted by timestamp

These mirror the sample files used by the Mucaro Scout app so instructors can load the same data directly from the lab `data/` directory.
